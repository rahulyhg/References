Integration with LinkedIn

To Integrate LinkedIn in your mobile application, you need to create a new application using LinkedIn Developer’s Account.

 

Step 1.1

Creating a new app in LinkedIn Developer account

Go to https://www.linkedin.com/developer/apps  and click on “Create Application” button. A form will open that needs to be filled for proceeding further.

 

 

Step 1.2

Set the Application Permission

Now, you need to set  the Default Application Permissions. And to do that, you have to select check box “r_basicprofile” and “r_emailaddress” and click on the “update” button to set the permission. We are selecting these check boxes as we will be requesting member’s permission to access their LinkedIn profile details and email address for your mobile application.

Step 1.3

Creating a new project

<p>    public void generateHashkey(){<br />
        try {<br />
            PackageInfo info = getPackageManager().getPackageInfo(<br />
                    PACKAGE,<br />
                    PackageManager.GET_SIGNATURES);<br />
            for (Signature signature : info.signatures) {<br />
                MessageDigest md = MessageDigest.getInstance(&quot;SHA&quot;);<br />
                md.update(signature.toByteArray());</p>
<p>                ((TextView) findViewById(R.id.package_name)).setText(info.packageName);<br />
                ((TextView) findViewById(R.id.hash_key)).setText(Base64.encodeToString(md.digest(),<br />
                                                                  Base64.NO_WRAP));<br />
            }<br />
        } catch (PackageManager.NameNotFoundException e) {<br />
            Log.d(TAG, e.getMessage(), e);<br />
        } catch (NoSuchAlgorithmException e) {<br />
            Log.d(TAG, e.getMessage(), e);<br />
        }<br />
    }</p>
